Snake Game

This project is a Snake Game that I originally built in my AP Computer Science class in High School. I built the base of the game using an online tutorial and later added more features in the Object Oriented Design & Programming class I took last spring.

Snake Game utilizes Java Swing and Awt to display the game and uses a two dimesional array to track the positions of the snake and apple.

Tic Tac Toe

This is a Tic-Tac-Toe game that I built in my Object Oriented Design & Programming class as an introduction to GUIs

It uses Java Swing and Awt to detect when buttons are pressed and stores the values of the spaces in a two dimesional array to detect when a player has won or it is a tie game.

Bank Managment System

This is a bank management system that I made for an assignment in my Object Oriented Design & Programming class. The purpose of this project was to get familiar with the ArrayList Data structures and exception handling.

It is terminal based system that asks the user various questions about opening and closing accounts, adding values to said accounts, and checking to see if they meet the requirements to have the accounts.
