This personal website was the first project in my Web Developement class. I was required to make a responsive and cohesive website that worked well on both desktop and mobile devices

This website utilizes the foundations framework for the basic division of content on a page and off canvas content.

Snake Game

This project is a Snake game that I originally built in my AP Computer Science class in high school. I created the base using an online tutorial and later added features in my Object-Oriented Design & Programming class.

The Snake game uses Java Swing and AWT for the UI and a two-dimensional array to track the positions of the snake and the apple.

Tic-Tac-Toe

This Tic-Tac-Toe game was built in my Object-Oriented Design & Programming class as an introduction to GUIs.

It uses Java Swing and AWT to detect button presses and stores the board state in a two-dimensional array to determine a win or a tie.
