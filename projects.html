Snake Game

This project is a Snake Game that I originally built in my AP Computer Science class in High School. I built the base of the game using an online tutorial and later added more features in the Intro To Object Oriented Programming class I took last spring.

Snake Game utilizes Java Swing and Awt to display the game and uses a two dimesional array to track the positions of the snake and apple.

Tic Tac Toe

This is a Tic-Tac-Toe game that I built in my Intro To Object Oriented Programming class as an introduction to GUIs

It uses Java Swing and Awt to detect when buttons are pressed and stores the values of the spaces in a two dimesional array to detect when a player has won or it is a tie game.
